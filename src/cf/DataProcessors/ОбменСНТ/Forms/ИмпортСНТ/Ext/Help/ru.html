Импорт электронных сопроводительных накладных на товары из XML файлов

Форма предназначена для создания новых и обновления существующих электронных сопроводительных накладных на товары  на основании XML файлов, выгруженных из Web-приложения ИС ЭСФ или Desktop-приложения ИС ЭСФ.

Кнопка "Загрузить из файлов" предназначена для выбора XML файлов электронных сопроводительных накладных на товары. При нажатии на кнопку "Загрузить из файлов" заполняется таблица "Импортируемые электронные сопроводительные накладные на товары". Возможна загрузка как входящих, так и исходящих электронных сопроводительных накладных на товары.

При загрузке выполняется поиск электронных сопроводительных накладных на товары в информационной базе, поиск осуществляется по значению реквизита "Идентификатор". Если электронная сопроводительная накладная на товары с идентификатором, указанным в импортируемом XML файле, будет найдена, то произойдёт обновление электронной сопроводительной накладной на товары, иначе будет создана новая электронная сопроводительная накладная на товары.

В поле "Структурная единица" указывается организация или структурное подразделение для которого загружаются XML файлы электронных сопроводительных накладных на товары.

Кнопка "Предварительный просмотр" предназначена для предварительного просмотра загружаемых электронных сопроводительных накладных на товары.

Кнопка "Открыть созданный СНТ" предназначена для просмотра электронной сопроводительной накладной на товары, находящегося в информационной базе.

Кнопка "Создать/обновить СНТ" предназначена для создания новых документов "Электронная сопроводительная накладная на товары" или обновления существующих документов "Электронная сопроводительная накладная на товары".
