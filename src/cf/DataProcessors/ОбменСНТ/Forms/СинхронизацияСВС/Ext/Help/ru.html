Получение электронных счетов-фактур из ИС ЭСФ

Форма предназначена для получения новых и измененных электронных сопроводительных накладных на товары через API ИС ЭСФ.

При нажатии на кнопку "Получить СНТ из ИС ЭСФ" будут получены:

  • Новые входящие СНТ.
  • Измененные входящие СНТ.
  • Измененные исходящие СНТ.
