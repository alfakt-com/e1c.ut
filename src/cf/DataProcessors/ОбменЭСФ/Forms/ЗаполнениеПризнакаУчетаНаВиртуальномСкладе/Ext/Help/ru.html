Заполнение реквизита "Ключ связи"

Обработка заполняет новый служебный реквизит "Ключ связи" в документах:

  • Реализация ТМЗ и услуг (табличная часть "ТМЗ"),
  • Счет-фактура (выданный) (табличная часть "ТМЗ"),
  • Перемещение ТМЗ (табличная часть "ТМЗ"),
  • Возврат ТМЗ от покупателя (табличная часть "ТМЗ"),
  • Списание ТМЗ (табличная часть "Товары"),
  • Комплектация ТМЗ (табличная часть "Комплектующие"),
  • Требование-накладная (табличная часть "Материалы").

Заполнение реквизита "Ключ связи" необходимо для обеспечения возможности указания источников происхождения для строк в вышеуказанных документах.
 
Реквизит ключ связи является служебным, поэтому не отображается в формах документов.
