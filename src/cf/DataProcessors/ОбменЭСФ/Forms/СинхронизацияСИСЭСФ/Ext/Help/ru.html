Получение электронных счетов-фактур из ИС ЭСФ

Форма предназначена для получения новых и измененных электронных счетов-фактур через API ИС ЭСФ.

При нажатии на кнопку "Получить электронные счета-фактуры из ИС ЭСФ" будут получены:

  • Новые входящие электронные счета-фактуры.
  • Измененные входящие электронные счета-фактуры.
  • Измененные исходящие электронные счета-фактуры.

Новые исходящие электронные счета-фактуры не будут получены, т.к. предполагается, что они были отправлены из данной информационной базы или из другой информационной базы и будут получены при выполнении обмена. Если требуется загрузить новый исходящий электронный счет-фактуру, то это можно выполнить с помощью формы "Импорт электронных счетов-фактур из XML файлов".
