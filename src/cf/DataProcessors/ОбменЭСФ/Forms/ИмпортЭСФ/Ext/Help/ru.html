Импорт электронных счетов-фактур из XML файлов

Форма предназначена для создания новых и обновления существующих электронных счетов-фактур на основании XML файлов, выгруженных из Web-приложения ИС ЭСФ или Desktop-приложения ИС ЭСФ.

Кнопка "Загрузить из файлов" предназначена для выбора XML файлов электронных счетов-фактур. При нажатии на кнопку "Загрузить из файлов" заполняется таблица "Импортируемые электронные счета-фактуры". Возможна загрузка как входящих, так и исходящих электронных счетов-фактур.

При загрузке выполняется поиск электронных счетов-фактур в информационной базе, поиск осуществляется по значению реквизита "Идентификатор". Если электронный счет-фактура с идентификатором, указанным в импортируемом XML файле, будет найден, то произойдёт обновление электронного счета-фактуры, иначе будет создан новый электронный счет-фактура.

В поле "Структурная единица" указывается организация или структурное подразделение для которого загружаются XML файлы электронных счетов-фактур.

Кнопка "Предварительный просмотр" предназначена для предварительного просмотра загружаемых электронных счетов-фактур.

Кнопка "Открыть созданный ЭСФ" предназначена для просмотра электронного счета-фактуры, находящегося в информационной базе.

Кнопка "Создать/обновить электронные счета-фактуры" предназначена для создания новых документов "Электронный счет-фактура" или обновления существующих документов "Электронный счет-фактура".
