Параметры подключения к серверу ИС ЭСФ

Форма предназначена для настройки параметров подключения к серверу ИС ЭСФ.

Параметры подключения к серверу ИС ЭСФ

Адрес - полный адрес сервера ИС ЭСФ, например: https://esf.gov.kz:8443/esf-web/ws/

Кнопка "Заполнить" предназначена для заполнения реквизитов "Защищенное соединение HTTPS", "Сервер", "Порт", "Ресурс", на основании введенного адреса.

Защищенное соединение HTTPS - Признак того, что необходимо работать с сервером ИС ЭСФ через защищенное HTTPS соединение.

Сервер - Имя сервера ИС ЭСФ, например: esf.gov.kz

Порт - Порт сервера ИС ЭСФ, например: 8443

Ресурс - Ресурс сервера ИС ЭСФ, например: esf-web/ws/

Сертификаты сервера ИС ЭСФ

Сертификаты сервера ИС ЭСФ необходимы для открытия защищенного HTTPS соединения с сервером ИС ЭСФ. Установка сертификатов происходит в файл cacert.pem, который располагается в каталоге исполняемых файлов текущей версии 1С:Предприятие, например: C:\Program Files\1cv82\8.2.19.68\bin

Гиперссылка "Установить сертификаты сервера ИС ЭСФ" предназначена для автоматической установки сертификатов.

Гиперссылка "Сохранить сертификаты сервера ИС ЭСФ" предназначена для сохранения сертификатов в файл, для последующей ручной установки.

Параметры прокси-сервера

Не использовать прокси-сервер - Доступ к серверу ИС ЭСФ будет осуществляться без использования прокси-сервера.

Использовать системные настройки прокси-сервера - Доступ к серверу ИС ЭСФ будет осуществляться через прокси-сервер, указанный в настройках операционной системы.

Использовать другие настройки прокси-сервера - Доступ к серверу ИС ЭСФ будет осуществляться через произвольный прокси-сервер.

Адрес сервера - Адрес произвольного прокси-сервера.

Порт - Порт произвольного прокси-сервера.

Пользователь - Пользователь для аутентификации на произвольном прокси-сервере.

Пароль - Пароль для аутентификации на произвольном прокси-сервере.
