Настройки обмена электронными счетами-фактурами

Способы обмена с ИС ЭСФ

Прямой обмен с ИС ЭСФ (Работа через API ИС ЭСФ) - Отправка и получение электронных счетов-фактур происходит непосредственно из информационной базы.

Обмен с ИС ЭСФ через выгрузку/загрузку XML файлов - Выданные электронные счет-фактуры подготавливаются в информационной базе, затем выгружаются в XML файлы, которые импортируются в ИС ЭСФ. Полученные электронные счета-фактуры экспортируются из ИС ЭСФ в XML файлы, которые загружаются в информационную базу.

Криптографическая библиотека

Для обмена электронными счетами-фактурами через API ИС ЭСФ требуется установить криптографическую библиотеку Национального удостоверяющего центра Республики Казахстан.

Библиотека предназначена для работы в операционной системе Microsoft Windows.

Если библиотека не будет установлена, то обмен электронными счетами-фактурами через API ИС ЭСФ будет невозможен.

Криптографическая бибилиотека может быть установлена:

  • На сервере 1С:Предприятие - Должна быть установлена на всех серверах 1С:Предприятие.
  • На компьютерах пользователей - Должна быть установлена на компьютерах всех пользователей, которые будут отправлять электронные счета-фактуры через API ИС ЭСФ.

Для обмена электронными счетами-фактурами через выгрузку/загрузку XML файлов установка библиотеки не требуется.

Подключение к ИС ЭСФ

Для получения и отправки электронных счетов-фактур через API ИС ЭСФ необходим доступ к серверу ИС ЭСФ.

Для обмена электронными счетами-фактурами через выгрузку/загрузку XML файлов доступ к серверу ИС ЭСФ не требуется.

Синхронизация по расписанию

При синхронизации с ИС ЭСФ в программу поступают:

  • Новые входящие электронные счета-фактуры.
  • Обновления существующих входящих электронных счетов-фактур.
  • Обновления существующих исходящих электронных счетов-фактур.

Автоматическая синхронизация выполняется только для тех профилей ИС ЭСФ, у которых:

  • Установлен флаг "Использовать для синхронизации с ИС ЭСФ".
  • У пользователя ИС ЭСФ задан пароль входа в ИС ЭСФ.

Синхронизация по расписанию доступна только при работе через API ИС ЭСФ.

Пользователи ИС ЭСФ

Для получения и отправки электронных счетов-фактур через API ИС ЭСФ, необходимо создать пользователей ИС ЭСФ.

Для обмена электронными счетами-фактурами через выгрузку/загрузку XML файлов создавать пользователей ИС ЭСФ не требуется.
