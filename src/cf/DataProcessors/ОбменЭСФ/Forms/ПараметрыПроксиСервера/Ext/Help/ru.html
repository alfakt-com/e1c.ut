Параметры прокси-сервера

Не использовать прокси-сервер - Доступ к серверу ИС ЭСФ будет осуществляться без использования прокси-сервера.

Использовать системные настройки прокси-сервера - Доступ к серверу ИС ЭСФ будет осуществляться через прокси-сервер, указанный в настройках операционной системы.

Использовать другие настройки прокси-сервера - Доступ к серверу ИС ЭСФ будет осуществляться через произвольный прокси-сервер.

Адрес сервера - Адрес произвольного прокси-сервера.

Порт - Порт произвольного прокси-сервера.

Пользователь - Пользователь для аутентификации на произвольном прокси-сервере.

Пароль - Пароль для аутентификации на произвольном прокси-сервере.
