Форма "Сопоставление виртуальных складов" - является помощником по получению и сопоставлению виртуальных складов со внутренними складами системы.

Предназначена для получения и сопоставления виртуальных складов со внутренними складами системы.

Процесс состоит из 3-ех этапов:

    1. Выбор организаций.
    2. Получение виртуальных складов.
    3. Сопоставление полученных виртуальных складов со внутренними складами системы.

Выбор организаций

  • С помощью флажков отметьте организации, для которых вы хотите сопоставить склады. По окончанию выбора организаций проследуйте ко второму шагу.

Примечание: В списке показаны только те организации, для которых настроены профили ИС ЭСФ.

Получение виртуальных складов по выбранным организациям

  • Для того чтобы получить виртуальные склады из ИС ЭСФ по выбранным организация, нажмите Получить виртуальные склады из ИС ЭСФ.
  • В списке будут показаны полученные виртуальные склады сгруппированные по выбранным организациям в шаге 1.

Примечание: В списке будут показаны все склады по выбранным организациям существующие в справочнике Виртуальные склады. При повторном использовании помощника, список будет заполнен из справочника при наличии полученных ранее виртуальных складов.

Сопоставление складов

  • Нажмите Заполнить (сопоставить), программа автоматически постарается сопоставить склады учетной системы с полученными виртуальными складами по наименованию.
  • Присутствует два вида сопоставления:
    • Склады с незаполненным вирт. складом - сопоставление будет выполнено только для тех строк табличной части в шаге 3, где не заполнена колонка "Виртуальный склад".
    • Все склады - сопоставление бдует выполнено для все строк, выбранные значения в колонке "Виртуальный склад" будут перезаполнены по результату сопоставления.
  • После автоматического сопоставления оставшиеся склады можно сопоставить вручную.
  • Для подтверждения нажмите Записать и закрыть.
