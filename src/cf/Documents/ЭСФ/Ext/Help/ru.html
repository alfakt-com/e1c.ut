Документ «Электронный счет-фактура»

Документ предназначен для выписки и получения счетов-фактур в электронном виде через Информационную систему по приему и обработке электронных счетов-фактур (ИС ЭСФ).

Выписка документа "Счет-фактура (выданный)" в электронном виде

При выписке документа "Счет-фактура (выданный)" в электронном виде, на основании документа "Счет-фактура (выданный)" создается документ "Электронный счет-фактура", который может быть отправлен получателю через ИС ЭСФ двумя способами:

  1. Отправка через API ИС ЭСФ
  2. Отправка через экспорт XML файла
Отправка через API ИС ЭСФ

Для отправки электронного счета-фактуры через API ИС ЭСФ необходимо нажать на кнопку "Отправить получателю через ИС ЭСФ". При нажатии на кнопку "Отправить получателю через ИС ЭСФ", открывается окно "Соединение с ИС ЭСФ", в котором необходимо указать пароль для входа в ИС ЭСФ (при наличии пароля в справочнике пользователя ИС ЭСФ, указание пароля не требуется). Далее открывается окно "Установка электронной подписи", в котором указывается сертификат ЭЦП и пароль к нему.

Обработка электронных счетов-фактур, отправленных в ИС ЭСФ, происходит в порядке очереди. Для получения результата обработки отправленного электронного счета-фактуры, можно воспользоваться кнопкой "Получить результат обработки ИС ЭСФ", либо результат обработки будет получен автоматически при следующей синхронизации с ИС ЭСФ.

Если отправленный электронный счет-фактура принят ИС ЭСФ, то электронному счету-фактуре присваивается регистрационный номер.

Отправка через экспорт XML файла

Для отправки электронного счета-фактуры через экспорт XML файла необходимо нажать на кнопку "Выгрузить в файл", после чего сохранить отправляемый электронный счет-фактуру в XML файл. Сохраненный XML файл можно импортировать в Desktop-приложение ИС ЭСФ или Web-приложение ИС ЭСФ, затем отправить импортированный XML файл получателю из Desktop-приложения ИС ЭСФ или Web-приложения ИС ЭСФ.

Чтобы отразить результат отправки электронного счета-фактуры, необходимо, из Desktop-приложения ИС ЭСФ или Web-приложения ИС ЭСФ, выгрузить отправленный электронный счет-фактуру, после чего загрузить его в информационную базу, нажав на кнопку "Загрузить из файлов", в форме списка электронных счетов-фактур.

Получение документа "Счет-фактура (полученный)" в электронном виде

Входящие электронные счета-фактуры могут быть созданы в информационной базе двумя способами:

  1. Получение через API ИС ЭСФ
  2. Получение через импорт XML файла

На основании входящих электронных счетов-фактур могут быть созданы документы "Счет-фактура (полученный)".

Получение через API ИС ЭСФ

Для получения документов "Электронный счет-фактура", через API ИС ЭСФ, необходимо нажать на кнопку "Получить из ИС ЭСФ", в форме списка документов "Электронный счет-фактура".

Получение через импорт XML файла
Для получения документов "Электронный счет-фактура", через импорт XML файла, необходимо выгрузить XML файлы электронных счетов-фактур из Desktop-приложения ИС ЭСФ или Web-приложения ИС ЭСФ. Затем нажать на кнопку "Загрузить из файлов", в форме списка документов "Электронный счет-фактура". 
Создание документа "Счет-фактура (полученный)"

На основании документов "Электронный счет-фактура" могут быть созданы документы "Счет-фактура (полученный)". Для этого необходимо выполнить команду из подменю "Создать на основании", либо нажать на гиперссылку "Документ не отражен в учете" в форме электронного счета-фактуры, на основании которого создается счет-фактура (полученный).

Отзыв отправленного электронного счета-фактуры

При отправке ошибочного электронного счета-фактуры, либо если была допущена ошибка в разделах B или С, существует возможность отозвать такой электронный счет-фактуру. Для отзыва необходимо нажать на кнопку "Отозвать в ИС ЭСФ".
