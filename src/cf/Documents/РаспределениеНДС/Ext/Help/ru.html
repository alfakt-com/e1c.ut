Распределение НДС

Документ формируется ежеквартально и предназначен для формирования регламентных операций по НДС:
Зачет по НДС - операция предназначена для определения суммы НДС, подлежащей уплате в бюджет за отчетный период. При выполнении операции учитываются данные о сумме налога, начисляемого за период и сумме налога, принятого в зачет по приобретенным товарам, услугам.
