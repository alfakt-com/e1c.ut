Электронный документ ВС

Данный документ является «контейнером» и предназначен для обмена данными между учетной системой налогоплательщика и модулем «Виртуальный склад», а так же для ведения учета товаров по источникам их происхождения (в том числе товаров, не подлежащих учету на ВС).

Документ формируется в конфигурации на основании учетных документов. ЭДВС может быть отправлен на сервер ИС «Виртуальный склад» средствами API, либо выгружен в XML-файл, для последующего импорта файла в систему ВС.
