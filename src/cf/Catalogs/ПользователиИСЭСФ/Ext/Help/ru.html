Справочник «Пользователи ИС ЭСФ»

Справочник предназначен для хранения  данных о пользователях ИС ЭСФ, от имени которых работают пользователи информационной базы.

В поле "Пользователь ИБ" указывается пользователь информационной базы, которому принадлежит данный пользователь ИС ЭСФ.

Кнопка "Выбрать сертификат" предназначена для выбора сертификата аутентификации, который будет использоваться для входа в ИС ЭСФ.

Кнопка "Сохранить сертификат" предназначена для сохранения сертификата аутентификации в файл.

В поле "Имя для входа в ИС ЭСФ" указывается имя пользователя ИС ЭСФ, обычно это ИИН пользователя указанный в сертификате аутентификации.

В таблице "Организации, от имени которых пользователь сможет выписывать и получать электронные счета-фактуры" отображаются профили ИС ЭСФ, от имени которох пользователь ИС ЭСФ сможет работать с ИС ЭСФ.

Кнопка "Проверить вход в ИС ЭСФ" позволяет проверить корректность введенных данных, выполнив вход в ИС ЭСФ под каждым профилем ИС ЭСФ, указанным в таблице "Организации, от имени которых пользователь сможет выписывать и получать электронные счета-фактуры".

Пароль для входа в ИС ЭСФ

Если пароль для входа в ИС ЭСФ указан, то он сохраняется в информационной базе и не запрашивается при каждом подключении к ИС ЭСФ.

Если пароль не указан, то при каждом подключении к ИС ЭСФ, будет запрашиваться пароль для входа в ИС ЭСФ.

Существует возможность сохранять пароль для входа в ИС ЭСФ на время сеанса работы пользователя, в этом случае пароль потребуется ввести один раз за все время сеанса работы пользователя.

Если используется автоматическое получение электронных счетов-фактур с помощью регламентного задания, то у пользователя ИС ЭСФ, от имени которого должны получаться электронные счета-фактуры, должен быть задан пароль для входа в ИС ЭСФ.
