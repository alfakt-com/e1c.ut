Метод оценки стоимости товаров


Для выбора доступны следующие методы оценки:

  • Средняя (взвешенная) за месяц;
  • ФИФО (взвешенная оценка);
  • ФИФО (скользящая оценка).

Метод ФИФО (скользящая оценка) доступен для выбора в случае, если включено использование партионного учета (раздел "Администрирование - Финансовый результат")

Система налогообложения

В прикладном решении поддерживается автоматизация деятельности организаций, использующих общую систему налогообложения и находящихся на упрощенной системе налогообложения. Используемый вариант указывается в поле Система налогообложения.

Для налога на добавленную стоимость (НДС) устанавливаются признаки:
- плательщик НДС
- применяется пропорциональный метод отнесения НДС в зачет

Резервы по сомнительным долгам


Флаг Формировать резервы по сомнительным долгам в бухгалтерском учете определяет необходимость ежемесячного отчисления в резервы по сомнительным долгам по дебиторской задолженности.

Флаг Формировать резервы по сомнительным долгам в налоговом учете определяет необходимость ежемесячного отчисления в резервы по сомнительным долгам по дебиторской задолженности.

Отражение расходов на оплату труда

Настройка определяет детализацию расчетов с работниками на счете 3350"Краткосрочная задолженность по оплате труда". Если флаг установлен, то документ Отражение зарплаты в финансовом учете должен содержать данные по работникам.

ТМЦ в эксплуатации

В случае установки флага "вести дополнительный учет на забалансовых счетах" появляется возможность весту учет ТМЦ на забалансовых счетах.
