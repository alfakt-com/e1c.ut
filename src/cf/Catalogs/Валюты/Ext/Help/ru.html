Валюты

Список предназначен для ведения валют, используемых в программе. Необходимые для работы валюты можно подобрать из Классификатора валют (рекомендуется) или добавить вручную.

Открыть список валют можно из раздела Справочники.

В списке выводится:

  • Наименование валюты - полное наименование валюты.
  • Цифр. код - уникальный числовой код, принят международным стандартом, служит для идентификации валюты при загрузке курсов из сети Интернет.
  • Симв. код - принят международным стандартом, используется как основное представление валюты.
  • Курс - значение курса на текущую дату.
  • Кратность - по умолчанию заполнено единицей и не выводится. Если курс какой-либо валюты очень низкий (точность составляет меньше четырех знаков дробной части), то в этом поле указывается количество котировочных единиц.

Например. Курс японского иена составляет 12,61 тенге за 10 иен. Тогда в поле "Кратность" указываем 10 (котировочная единица японского иена), а в поле "Курс" 12.61.

Добавление валюты из классификатора валют

  • Нажмите Подобрать из классификатора (рекомендуется для корректного ввода валюты).
  • Следующие поля новой валюты будут заполнены из классификатора автоматически: Наименование, Символьный код, Цифровой код, положение переключателя Курс валюты.

Ввод валюты не из классификатора (вручную)
  • Нажмите Создать.
  • В ответ на вопрос программы о подборе из классификатора нажмите Нет.
  • Заполните сведения о валюте.

Загрузка курсов валют
  • Нажмите Загрузить курсы валют для перехода к загрузке курсов валют с сайта.
  • Также в программе предусмотрено регламентное задание Загрузка курсов валют для автоматической загрузки курсов валют с сайта Нац.банка РК.
