Справочник «Профили ИС ЭСФ»

Справочник предназначен для хранения данных о профилях пользователей ИС ЭСФ.

Каждый пользователь ИС ЭСФ может иметь несколько профилей, например, пользователь ИС ЭСФ "Иванов И.И." может иметь профили: "Иванов И.И. (физическое лицо)", "Иванов И.И. (сотрудник ТОО МИР)", "Иванов И.И. (сотрудник ТОО Вишенка)", т.е. один пользователь ИС ЭСФ может отправлять и получать электронные счета-фактуры от имени различных организаций.

В поле "Пользователь ИС ЭСФ" указывается пользователь ИС ЭСФ, которому принадлежит данный профиль ИС ЭСФ.

В поле "Структурная единица" указывается организация или подразделение организации, для которых создается профиль ИС ЭСФ.

Если флаг "Получать электронные счета-фактуры из ИС ЭСФ" установлен, то данный профиль будет отображаться в форме "Получение электронных счетов-фактур из ИС ЭСФ".

Если у профиля ИС ЭСФ установлен флаг "Получать электронные счета-фактуры из ИС ЭСФ" и для пользователя ИС ЭСФ, который является владельцем профиля, установлен пароль входа в ИС ЭСФ, то для данного профиля ИС ЭСФ будет выполняться автоматическое получение новых и измененных электронных счетов-фактур с помощью регламентного задания.
